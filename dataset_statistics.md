## 四、关键发现

1. **文件数量差异**: 单变量版本(870个)比多变量版本(200个)多约4.4倍
2. **序列长度差异**: 多变量版本平均长度(107,760)是单变量版本(38,814)的约2.8倍
3. **数据格式差异**:
   - TSB-AD-U: 多变量数据集被展平为多个单变量文件
   - TSB-AD-M: 保持原始多变量格式，每文件含多个通道
4. **论文实验对应**:
   - Table 1 (单变量结果) → 使用 TSB-AD-U
   - Table 6 (多变量结果) → 使用 TSB-AD-M
